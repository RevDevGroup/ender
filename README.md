# Ender - SMS Gateway con MQTT y Celery

Saas for sending SMS in Cuba :cuba:

Esta implementación utiliza MQTT para la comunicación con dispositivos y Celery para el procesamiento asíncrono, eliminando la dependencia de direcciones IP.
