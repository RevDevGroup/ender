Usage Guidelines

Primary Color (Black/White)
  • Buttons and interactive elements
  • Focus rings and outlines
  • Important UI actions
Accent Color (Orange #e94d1f)
  • Decorative dots and circles
  • Hover states on links
  • Visual highlights and badges
  • Chart primary color
  • Code syntax highlighting (keywords)
Typography
  • Libre Baskerville: Headings, titles, card titles
  • Inter: Body text, buttons, labels, UI elements
  • JetBrains Mono: Code blocks, inline code, technical content
